# SkyGuard Error: SCHEMA_VALIDATION

## תיאור השגיאה
שגיאה זו מתרחשת כאשר ערך מספרי (כגון גובה או עוצמת אות) חורג מהגבולות שהוגדרו בסכמה של Zod, אך סוג המשתנה (Number) תקין.

## סיבות נפוצות (Root Cause)
1. **Sensor Glitch (רעש חיישנים):** קפיצות מתח במכ"ם שגורמות לקריאת ערך קיצון (למשל `signal_strength: 999`).
2. **Calibration Needed:** המטוס טס בגובה תקין, אך החיישן במטוס לא מכוייל ומדווח גובה שלילי (`altitude: -500`) שלא מעל הים.
3. **Buggy Firmware:** גרסת קושחה חדשה במטוס שולחת נתונים בסקאלה שונה (למשל מטרים במקום רגל).
